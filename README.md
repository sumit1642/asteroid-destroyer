## Features

- Precise, Realistic Controls
- Fast-Paced, Intense Combat
- Progressively Tougher
- Stunning Visuals
- Challenging Enemies

## Controls

| Action     | Key     |
| ---------- | ------- |
| Move Up    | `W`     |
| Move Down  | `S`     |
| Move Left  | `A`     |
| Move Right | `D`     |
| Shoot      | `Space` |
